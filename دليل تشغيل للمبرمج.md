# 🚀 دليل تشغيل موقع "اتعلم بالعربي" — للمبرمج

---

## 📋 المتطلبات الأولية (لازم تكون مثبتة)

| برنامج | الإصدار المطلوب | رابط التحميل |
|--------|----------------|-------------|
| PHP | 8.2 أو أحدث | https://www.php.net |
| Composer | أي إصدار | https://getcomposer.org |
| Node.js | 18 أو أحدث | https://nodejs.org |
| MySQL | 8.0 أو أحدث | https://dev.mysql.com |

---

## 🗄️ الخطوة 1 — إعداد قاعدة البيانات

افتح MySQL وشغّل:

```sql
CREATE DATABASE et3alem_araby CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

---

## ⚙️ الخطوة 2 — إعداد الـ Backend (Laravel)

افتح Terminal وادخل على فولدر الـ backend:

```bash
cd backend
```

### نسخ ملف الإعدادات:
```bash
cp .env.example .env
```

### تثبيت الـ packages:
```bash
composer install
```

### توليد مفتاح التشفير:
```bash
php artisan key:generate
```

### فتح ملف `.env` وتعديل بيانات قاعدة البيانات:
```
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=et3alem_araby
DB_USERNAME=root
DB_PASSWORD=كلمة_مرورك_هنا
```

### تشغيل الـ migrations (إنشاء الجداول):
```bash
php artisan migrate
```

### إنشاء حساب الأدمن الأول:
```bash
php artisan db:seed
```

### إنشاء رابط Storage:
```bash
php artisan storage:link
```

---

## 🎨 الخطوة 3 — إعداد الـ Frontend (React)

افتح Terminal جديد وادخل على فولدر الـ frontend:

```bash
cd "موقع اتعلم بل العربي"
```

### تثبيت الـ packages:
```bash
npm install
```

### نسخ ملف الإعدادات:
```bash
cp .env.example .env.local
```

### فتح `.env.local` وتحديد عنوان الـ API:
```
VITE_API_URL=http://127.0.0.1:5000/api
```

---

## ▶️ الخطوة 4 — تشغيل المشروع

### تشغيل الـ Backend (في Terminal 1):
```bash
cd backend
php artisan serve --port=5000
```

### تشغيل الـ Frontend (في Terminal 2):
```bash
cd "موقع اتعلم بل العربي"
npm run dev
```


## 🐛 حل المشاكل الشائعة

### مشكلة: `Storage permission denied`
```bash
chmod -R 775 backend/storage
chmod -R 775 backend/bootstrap/cache
```

### مشكلة: `Class not found`
```bash
cd backend
composer dump-autoload
```

### مشكلة: الـ frontend مش بيوصل للـ API
- تأكد إن `VITE_API_URL` في `.env.local` صح
- تأكد إن الـ backend شغّال على بورت 5000

### مشكلة: قاعدة البيانات فارغة بعد الـ migrate
```bash
cd backend
php artisan db:seed --force
```
