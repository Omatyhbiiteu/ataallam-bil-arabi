## 📖 نظرة عامة

**اتعلم بالعربي Pro** هي منصة تعليمية متقدمة مصممة لجعل تعلم اللغات تجربة ممتعة وفعّالة. تجمع المنصة بين أحدث تقنيات الذكاء الاصطناعي والتصميم الحديث لتقديم تجربة تعليمية فريدة من نوعها.

### 🎯 لماذا اتعلم بالعربي Pro؟

- **📚 قصص تفاعلية**: تعلم من خلال قصص مشوقة مصممة بعناية لكل مستوى
- **🧠 نظام SRS ذكي**: نظام تكرار متباعد (Spaced Repetition System) لتحسين الحفظ طويل الأمد
- **🎤 نطق بالذكاء الاصطناعي**: استمع للنطق الصحيح بصوت طبيعي باستخدام AI
- **📊 تتبع التقدم**: راقب تطورك وإنجازاتك بشكل مرئي وتفاعلي
- **🎨 واجهة عصرية**: تصميم سلس وجذاب مع دعم الوضع الليلي/النهاري
- **🔐 لوحة تحكم متقدمة**: إدارة شاملة للمحتوى والمستخدمين والإعدادات

## 🚀 التشغيل المحلي

### المتطلبات الأساسية

قبل البدء، تأكد من تثبيت:

- **Node.js** (الإصدار 16 أو أحدث) - [تحميل](https://nodejs.org/)
- **npm** (يأتي مع Node.js)
- محرر أكواد (يُنصح بـ [VS Code](https://code.visualstudio.com/))

### خطوات التثبيت

1. **استنساخ المشروع** (إذا كان على Git):
   ```bash
   git clone <repository-url>
   cd et3alem-bel-araby
   ```

2. **تثبيت الحزم**:
   ```bash
   npm install
   ```

3. **إعداد المتغيرات البيئية**:
   
   أنشئ ملف `.env.local` في المجلد الرئيسي وأضف:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```
   
   > 💡 للحصول على مفتاح Gemini API، قم بزيارة [Google AI Studio](https://ai.studio/)

4. **تشغيل التطبيق**:
   ```bash
   npm run dev
   ```

5. **فتح المتصفح**:
   
   افتح [http://localhost:5173](http://localhost:5173) في متصفحك

### أوامر إضافية

```bash
# بناء النسخة الإنتاجية
npm run build

# معاينة النسخة الإنتاجية
npm run preview
```

## 📚 التوثيق

### الوصول للوحة التحكم

1. قم بتشغيل التطبيق
2. انقر على أيقونة الإعدادات (⚙️) في الزاوية العلوية
3. أدخل رمز الحماية الموجود في `VITE_ADMIN_PASSWORD` داخل ملف `.env`
4. ستظهر لك لوحة التحكم الكاملة

### إضافة قصة جديدة

1. افتح لوحة التحكم
2. اذهب إلى تبويب "القصص المنشورة"
3. اضغط على "قصة جديدة"
4. املأ البيانات:
   - عنوان القصة
   - المستوى (مبتدئ/متوسط/متقدم/خبير)
   - صورة القصة
   - وصف قصير
   - المحتوى (دعم Markdown)
   - الترجمة (اختياري)
5. احفظ القصة
6. أضف الأسئلة من خلال "أسئلة القصة"

### إدارة المنهج

1. اذهب إلى تبويب "المنهج التعليمي"
2. أضف وحدة جديدة (Module)
3. داخل كل وحدة، أضف دروس متعددة
4. لكل درس، يمكنك إضافة محتوى وأسئلة
