# روابط فتح المشروع وتشغيله

هذا هو الملف السريع الذي يقرأه أي مبرمج أو AI لتشغيل المشروع.

المشروع يعمل من 3 ترمينالات منفصلة:

- Terminal 1: Laravel API
- Terminal 2: موقع المستخدم
- Terminal 3: لوحة التحكم

> مهم: لا تغيّر إعدادات React في التطوير. اترك `VITE_BACKEND_API_URL=/api` لأن Vite يوصّل الطلبات إلى Laravel تلقائياً ويمنع مشاكل CORS.

---

## قبل التشغيل

افتح كل ترمينال من مجلد المشروع الرئيسي.

المقصود بمجلد المشروع الرئيسي هو المجلد الذي بداخله:

```text
backend
موقع اتعلم بل العربي
لوحه التحكم الخارجيه
```

---

## Terminal 1 — تشغيل الباك إند

```bash
cd backend
composer install
if [ ! -f .env ]; then cp .env.example .env; fi
if ! grep -q '^APP_KEY=base64:' .env 2>/dev/null; then php artisan key:generate --ansi; fi
mkdir -p database
if [ ! -f database/database.sqlite ]; then touch database/database.sqlite; fi
php artisan migrate --force
php artisan serve --host=127.0.0.1 --port=5000
```

اختبار الباك إند:

```text
http://127.0.0.1:5000/api/health
```

---

## Terminal 2 — تشغيل موقع المستخدم

```bash
cd "موقع اتعلم بل العربي"
npm install
if [ ! -f .env ]; then cp .env.example .env; fi
npm run dev -- --host 127.0.0.1 --port 3000
```

رابط موقع المستخدم:

```text
http://127.0.0.1:3000/
```

---

## Terminal 3 — تشغيل لوحة التحكم

```bash
cd "لوحه التحكم الخارجيه"
npm install
if [ ! -f .env ]; then cp .env.example .env; fi
npm run dev -- --host 127.0.0.1 --port 3001
```

رابط لوحة التحكم:

```text
http://127.0.0.1:3001/admin/
```

بيانات دخول الأدمن الافتراضية:

```text
email: admin@et3alem.local
password: قيمة ADMIN_BOOTSTRAP_PASSWORD داخل backend/.env
```

غيّر `ADMIN_BOOTSTRAP_PASSWORD` و`VITE_ADMIN_PASSWORD` قبل تسليم المشروع أو نشره.

---

## لو ظهر خطأ اتصال

تأكد أن Terminal 1 شغال وأن هذا الرابط يرجع JSON:

```text
http://127.0.0.1:5000/api/health
```

لو Laravel شغال على منفذ 8000 بدل 5000، الواجهات غالباً ستتصل به تلقائياً، لكن الأفضل للتشغيل السلس استخدام 5000 كما في الأوامر فوق.

---

## روابط الفتح النهائية

- موقع المستخدم: `http://127.0.0.1:3000/`
- لوحة التحكم: `http://127.0.0.1:3001/admin/`
- اختبار الباك إند: `http://127.0.0.1:5000/api/health`
